
### What is this

    Personal piano recordings. Structure for [backand](https://github.com/backand/)

### Getting started

    $ npm install
    $ bower install
    $ gulp serve

### Deploy

    $ sudo npm install -g backand  
    $ gulp build:dist  
    $ backand sync --app pakpakpiano --master *** --user *** --folder /path/to/project/folder
